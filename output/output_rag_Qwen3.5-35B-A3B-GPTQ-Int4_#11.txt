1. **Effective Core Hour Remaining** adalah sisa kuota core hour efektif yang menjadi dasar Slurm untuk menjalankan job. Nilainya berubah tergantung status job:
   - Ketika **tidak ada job berjalan**, Effective Core Hour Remaining sama dengan sisa kuota **Actual Core Hour**.
   - Ketika **ada job berjalan**, Effective Core Hour Remaining sama dengan sisa kuota **Total Core Hour** (yang sudah dikurangi Booked Core Hour).

2. **Actual Core Hour Usage** adalah penggunaan core hour yang sesungguhnya (realtime) yang dihitung berdasarkan waktu job benar-benar berjalan.
   - Rumus penghitungannya:
     - **ALELEON CPU Core Hour (CCH)** = Slurm Core Minutes / 60
     - **ALELEON GPU Hour (GH)** = Slurm GPU Minutes / 60
   - Actual Core Hour Usage menjadi **basis biaya** penggunaan ALELEON.

Secara ringkas, **Actual Core Hour** hanya menghitung waktu yang sudah terpakai, sedangkan **Effective Core Hour** adalah sisa kuota yang tersedia untuk menjalankan job baru, yang bisa berkurang lebih cepat jika ada job yang sedang berjalan karena memperhitungkan alokasi waktu masa depan (Booked Core Hour).
